SxLib WinForm Functions

SxLib for WinForms-based UIs.

sxlib.Specialized.SxLibWinForms.SynLoadDelegate

A delegate for a callback from SxLib.Load().

NameDescription
Event:A enum containing states of Synapse X loading.
Param:An extra parameter used for some events.

sxlib.Specialized.SxLibWinForms.LoadEvent

This event will be periodically called back during Synapse X initialization. Use SxLib.Load() to start initialization.

For a list of load events, see here.

Example

SxLibWinForms SxWinForms;
//...
SxWinForms.LoadEvent += LoadEventHandler;
SxWinForms.Load();

async void LoadEventHandler(SxLibBase.SynLoadEvents LoadEvent, object Param) 
{
    switch (LoadEvent)
    {
        case SxLibBase.SynLoadEvents.NOT_LOGGED_IN:
            //User is not logged in
            break;
        case SxLibBase.SynLoadEvents.READY:
            //...
            break;
        case SxLibBase.SynLoadEvents...:
            //Full list of load events can be found above.
            break;
        default:
            break;
    }
}

sxlib.Specialized.SxLibWinForms.SynAttachDelegate

A delegate for a callback from SxLib.Attach().

NameDescription
Event:A enum containing states of Synapse X attaching.
Param:An extra parameter used for some events.

sxlib.Specialized.SxLibWinForms.AttachEvent

This event will be periodically called back during Synapse X attaching. Use SxLib.Attach() to start the attaching process.

For a list of attach events, see here.

Example

SxLibWinForms SxWinForms;
//...
SxWinForms.AttachEvent += AttachEventHandler;
SxWinForms.Attach();

async void AttachEventHandler(SxLibBase.SynAttachEvents AttachEvent, object Param)
{
    switch (LoadEvent)
    {
        case SxLibBase.SynAttachEvents.FAILED_TO_FIND:
            //Failed to find Roblox
            break;
        case SxLibBase.SynAttachEvents.READY:
            //...
            break;
        case SxLibBase.SynAttachEvents...:
            //Full list of attach events can be found above.
            break;
        default:
            break;
    }
}

sxlib.Specialized.SxLibWinForms.Load

This will start the initial loading process of Synapse X. You must attach a handler to 'LoadEvent' to get callback events from this function.

Returns: If initial loading fails, this will return false.

Example can be found here.

sxlib.Specialized.SxLibWinForms.Close

Call this when your application exits.

sxlib.Specialized.SxLibWinForms.Attach

This will start the initial attaching process of Synapse X. You must attach a handler to 'AttachEvent' to get callback events from this function.

Returns: If initial attaching fails, this will return false.

Example can be found here.

sxlib.Specialized.SxLibWinForms.ScriptHub

This will start the script hub. You must attach a handler to 'ScriptHubEvent' to get the actual script hub contents from this function.

Returns: If the script hub is already open, this will return false. Use SxLib.ScriptHubMarkAsClosed() to allow the script hub to be open again.

For more, see here.

sxlib.Specialized.SxLibWinForms.Ready

This will return if Synapse X can execute scripts at the current time.

Returns: If Synapse X is fully loaded and can execute scripts.

sxlib.Specialized.SxLibWinForms.Execute(string Script)

This will execute a script in Synapse X. Note if you try to call this function without being attached, you will get a 'NOT_ATTACHED' event from the AttachEvent callback.

NameDescription
Script:The script to be executed.

Example

SxLibWinForms SxWinForms;
//...
SxWinForms.Execute("print(\"Hello, World!\"");

sxlib.Specialized.SxLibWinForms.ScriptHubMarkAsClosed

This will mark the Script Hub as closed, allowing you to call SxLib.ScriptHub() again.

sxlib.Specialized.SxLibWinForms.GetOptions

This will return the current options the user has selected.

Returns: The user's options.

sxlib.Specialized.SxLibWinForms.SetOptions(sxlib.Static.Data.Options Options)

This will set the options of the current user.

NameDescription
Options:The he options to set.

sxlib.Specialized.SxLibWinForms.SetWindow(Form _Current)

Set the Window currently in use. You must call this every time you create a new 'main' window.

NameDescription
_Current:The current window.
