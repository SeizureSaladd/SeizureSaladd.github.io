Installing SxLib

To use SxLib, you must add it as a refrence to your Visual Studio (or whatever IDE) project. If you don't know how to do this, then follow the instructions below. If you don't have Visual Studio, then download it here. Also make sure you aren't downloading Visual Studio Code.

First, if you haven't already, create a Visual Studio project.

Next, choose the UI framework of your choosing. SxLib has support for both WPF and WinForms.

Next, configure the name and other additional information and click Next.

If you chose a non-.NET Framework project template, you'll be asked to select your .NET version. Select the version of your choice and click Create.

When brought into Visual Studio, right click on References and click Add Reference in the Solution Explorer.

Once in the Reference Manager, click Browse....

Locate sxlib.dll in your Synapse X folder and add it as a reference.

Finally, press OK in the Reference Manager to finish adding SxLib as a reference. You can now move on to setup to set up SxLib.
