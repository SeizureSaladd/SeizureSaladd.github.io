SxLibBase

sxlib.Specialized.SxLibBase.SynLoadEvents

Loading events for Synapse (will be sent as callbacks with the 'Load' function)

Load eventsDescription
sxlib.Specialized.SxLibBase.SynLoadEvents.UNKNOWNUnknown error. You should usually exit your application if this is received.
sxlib.Specialized.SxLibBase.SynLoadEvents.NOT_LOGGED_INThe user is not logged in to Synapse X. SxLib does not include methods to log a user in at the present moment, you must tell the user to login via the official UI.
sxlib.Specialized.SxLibBase.SynLoadEvents.NOT_UPDATEDSynapse X is currently not updated.
sxlib.Specialized.SxLibBase.SynLoadEvents.FAILED_TO_VERIFYSynapse X detected a discrepancy within its data. You should usually exit your application if this is received.
sxlib.Specialized.SxLibBase.SynLoadEvents.FAILED_TO_DOWNLOADSynapse X failed to download needed UI files. This is usually caused by anti-virus software.
sxlib.Specialized.SxLibBase.SynLoadEvents.UNAUTHORIZED_HWIDUser is not whitelisted for Synapse X.
sxlib.Specialized.SxLibBase.SynLoadEvents.ALREADY_EXISTING_WLThis is a rare error - tell the user to use the official UI to fix this issue.
sxlib.Specialized.SxLibBase.SynLoadEvents.NOT_ENOUGH_TIMEThere was not enough time preceding the last whitelist change. Whitelist changes can only happen every 24 hours.
sxlib.Specialized.SxLibBase.SynLoadEvents.CHECKING_WLThe 'Checking Whitelist' phase of Synapse X initialization.
sxlib.Specialized.SxLibBase.SynLoadEvents.CHANGING_WLThe 'Changing Whitelist phase of Synapse X initialization. This won't occur as often, only if the user requires a whitelist change. You might get an error code after this.
sxlib.Specialized.SxLibBase.SynLoadEvents.DOWNLOADING_DATAThe 'Downloading Data' phase of Synapse X initialization.
sxlib.Specialized.SxLibBase.SynLoadEvents.CHECKING_DATAThe 'Checking Data' phase of Synapse X initialization.
sxlib.Specialized.SxLibBase.SynLoadEvents.DOWNLOADING_DLLSThe 'Downloading DLLs' phase of Synapse X initialization.
sxlib.Specialized.SxLibBase.SynLoadEvents.READYSynapse X has loaded successfully and can now be used.

sxlib.Specialized.SxLibBase.SynAttachEvents

Attaching events for Synapse (will be sent as callbacks with the 'Attach' function)

Attach eventDescription
sxlib.Specialized.SxLibBase.SynAttachEvents.CHECKINGThe 'Checking' phase of Synapse X attaching. Note that this event may be sent if the user has Auto-Attach or Auto-Launch enabled, even if you did not invoke the SxLib.Attach() method.
sxlib.Specialized.SxLibBase.SynAttachEvents.INJECTINGThe 'Injecting' phase of Synapse X attaching.
sxlib.Specialized.SxLibBase.SynAttachEvents.CHECKING_WHITELISTThe 'Checking Whitelist' phase of Synapse X attaching.
sxlib.Specialized.SxLibBase.SynAttachEvents.SCANNINGThe 'Scanning' phase of Synapse X attaching. Note this event is also sent in teleports as well.
sxlib.Specialized.SxLibBase.SynAttachEvents.READYSynapse X has successfully attached.
sxlib.Specialized.SxLibBase.SynAttachEvents.FAILED_TO_ATTACHSynapse X has failed to attach - generic error.
sxlib.Specialized.SxLibBase.SynAttachEvents.FAILED_TO_FINDSynapse X has failed to find Roblox.
sxlib.Specialized.SxLibBase.SynAttachEvents.NOT_RUNNING_LATEST_VER_UPDATINGSynapse X has detected they are not running the latest version of Synapse X. Synapse will now update itself.
sxlib.Specialized.SxLibBase.SynAttachEvents.UPDATING_DLLSThis is if Synapse X is updating itself and is currently updating its DLLs.
sxlib.Specialized.SxLibBase.SynAttachEvents.NOT_UPDATEDThis is if Synapse X has detected an update, is not released yet.
sxlib.Specialized.SxLibBase.SynAttachEvents.FAILED_TO_UPDATEThis is if Synapse X auto-update failed.
sxlib.Specialized.SxLibBase.SynAttachEvents.REINJECTINGThis event is called after Synapse X updates, and also specifies 'Param' as a string (the new version string)
sxlib.Specialized.SxLibBase.SynAttachEvents.NOT_INJECTEDThis event is sent if you attempt to use the SxLib.Execute() method without being attached.
sxlib.Specialized.SxLibBase.SynAttachEvents.ALREADY_INJECTEDThis event is sent if you attempt to use the SxLib.Attach() method while already being attached.
sxlib.Specialized.SxLibBase.SynAttachEvents.PROC_CREATIONThis event is sent if Synapse X detects a new Roblox process being created.
sxlib.Specialized.SxLibBase.SynAttachEvents.PROC_DELETIONThis event is sent if Synapse X detects a Roblox process being deleted.
